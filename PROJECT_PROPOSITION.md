DigiAssistant is a simple tool for digital maturity checks. It replaces long questionnaires with a short conversation. You answer questions in a chat and the system scores your level in real time.

Project focus:

Collect basic company info.

Run a 72-step diagnostic through a chat interface.

Score each answer from 0 to 3.

Show progress.

Save the session so you return later.


Pages:

Dashboard. Form for company data and a button to start the diagnostic.

Diagnostic. Chat page with questions, answers, scores, and progress.

Results. Final score, scores for the six dimensions, gaps, and recommendations.


Tech stack:

Backend with FastAPI and MongoDB.

AI with Gemini or OpenAI.

Frontend with React and Vite.

Deployment on Railway and Vercel.


Extra features:

PDF and JSON export.

Clean UI.

Good error handling.


Goal: Give companies a quick view of their digital level so they take action fast.
